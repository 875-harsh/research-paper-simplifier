# 📄 Research Paper Simplifier

An AI-powered web application that simplifies complex research papers into easy-to-understand summaries using Natural Language Processing (NLP).

---

## 🚀 Features

* 📂 Upload research papers in PDF format
* 🧠 Automatic text summarization using NLP models
* 🔍 Keyword extraction for better understanding
* ⚡ Fast and user-friendly interface

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Hugging Face Transformers
* PyMuPDF

---

## 📦 Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/875-harsh/research-paper-simplifier.git
```

### 2️⃣ Navigate to project folder

```bash
cd research-paper-simplifier
```

### 3️⃣ Create virtual environment (optional but recommended)

```bash
python -m venv venv
venv\Scripts\activate
```

### 4️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```
